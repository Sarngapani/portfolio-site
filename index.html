I'm a MuleSoft Developer at Coforge Limited, specializing in designing and deploying enterprise integration solutions for the banking domain. My work spans the full API lifecycle — from gathering requirements to production support.

I work with API-Led Connectivity patterns — Experience, Process, and System layers — that allow organizations to modernize without disrupting critical operations. I've migrated legacy Spring applications to Mule 4, containerized APIs with Docker, and deployed on Kubernetes from scratch.

I believe good integration engineering is like plumbing — when done right, nobody notices it. When done poorly, everything floods.
